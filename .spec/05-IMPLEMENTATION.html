§8 风险矩阵

风险与缓解措施

风险等级缓解措施
servo JSContext 与 Bao host functions 注册兼容性中Phase 2 优先验证 host functions 在 servo JSContext 中注册可行；servo 和 Bao 共用 mozjs 0.15.14
mozjs 缺少 Debugger API 绑定低Debugger API 是 JS 层的，不需要 Rust 绑定；直接在 SM 内执行 JS Debugger 代码
Bun 上游合并冲突低内部保留 bun_* 命名，JSC 层独立维护，diff 最小化
SpiderMonkey GC 模型差异中mozjs crown lint 编译时检查 rooting 正确性；RAII 封装自动管理
servo API 不稳定中锁定 servo 版本；ServoDelegate/WebViewDelegate 变化时最小化适配
Playwright CDP 兼容性中优先实现 Playwright 核心流程所需的 Domain；参考 Firefox CDP 实现
Canvas 噪声被检测低噪声在 servo webrender 层注入，JS 层无法检测；确定性种子保证一致性
